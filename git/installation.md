## Mac
### Let's use [homebrew](http://brew.sh/)
It's easy. Just:

```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
... in the [terminal](../terminal/terminal.md).

then: `brew install git`
